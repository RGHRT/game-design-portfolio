# RPG 角色成长与关卡难度数值表

## 项目简介

本项目用于展示基础数值策划能力。

我设计了一套简化的 RPG 角色成长与关卡难度数值表，用于模拟角色属性成长、经验消耗、敌人强度、伤害公式、关卡推荐等级和资源产出消耗之间的关系。

这不是某个具体商业游戏的真实数值，而是一个用于作品集展示的虚构数值模型。它的重点不在于“数值一定完美”，而在于展示我能够使用 Excel 建立公式驱动的数值表，并从成长节奏、资源压力和关卡体验角度进行分析。

## 表格结构

### 0_设计说明

说明本数值表的设计目的、核心思路和主要参数。

包括基础生命值、攻击力、防御力、成长率、经验成长率、伤害保底比例等内容。

### 1_角色属性成长

设计角色 1—80 级的属性成长，包括：

- 生命值 HP
- 攻击力 ATK
- 防御力 DEF
- 战力评分
- 升级所需经验
- 累计经验

这一页主要用于观察角色随等级提升时的成长速度，避免出现前期成长过慢或后期数值膨胀过快的问题。

### 2_经验消耗曲线

观察不同等级阶段的经验消耗变化。

前期升级成本较低，用于提供快速成长反馈；中后期经验消耗逐步提高，用于制造资源规划压力，并延长玩家的长期培养目标。

### 3_敌人属性设计

设计普通怪、精英怪和 Boss 的属性成长，包括：

- 敌方生命值
- 敌方攻击力
- 敌方防御力

通过不同敌人倍率，形成普通怪、精英怪、Boss 之间的强度层级。
