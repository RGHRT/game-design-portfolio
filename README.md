# 赵锦城｜游戏策划 / 数值策划作品集

## 个人简介

我目前就读于武汉大学遥感科学与技术专业，具备数据分析、数学建模、Python 与 Excel 基础。

我长期关注游戏系统、数值成长、资源循环与玩家体验，希望从数值策划或系统策划方向进入游戏行业。

我是《明日方舟》开服玩家，也持续体验《明日方舟：终末地》；同时《无畏契约》累计游玩 600+ 小时。相比单纯“喜欢玩游戏”，我更希望从策划视角去理解游戏如何通过系统、数值、关卡、资源和版本节奏影响玩家体验。

本仓库用于整理我的游戏策划相关作品，包括系统分析、资源规划分析、数值表设计与玩家体验观察。
