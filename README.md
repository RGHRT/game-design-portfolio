# 赵锦城｜游戏策划 / 数值策划作品集

## 个人简介

我目前就读于武汉大学遥感科学与技术专业，具备数据分析、数学建模、Python 与 Excel 基础。

我长期关注游戏系统、数值成长、资源循环与玩家体验，希望从数值策划或系统策划方向进入游戏行业。

我是《明日方舟》开服玩家，也持续体验《明日方舟：终末地》；同时《无畏契约》累计游玩 600+ 小时。相比单纯“喜欢玩游戏”，我更希望从策划视角去理解游戏如何通过系统、数值、关卡、资源和版本节奏影响玩家体验。

本仓库用于整理我的游戏策划相关作品，包括系统分析、资源规划分析、数值表设计与玩家体验观察。

## 作品目录

### 01. 《明日方舟》资源规划与长期养成系统分析

从理智、材料、龙门币、经验书、精英化、技能专精、模组等角度，分析《明日方舟》如何通过有限资源制造长期目标，并讨论资源压力、玩家规划行为与长期留存之间的关系。

### 02. 基于《明日方舟一图流》的玩家资源决策分析

[查看分析初稿](./02_明日方舟一图流分析/基于明日方舟一图流的玩家资源决策分析.md)

基于《明日方舟一图流》工具站，分析关卡推荐、商店性价比、材料价值表、攒抽计算器等功能背后的玩家需求，观察外部工具如何帮助玩家理解复杂资源系统。

### 03. RPG 角色成长与关卡难度数值表

自制一套 RPG 角色成长与关卡难度数值表，包含角色属性成长、经验消耗、敌人属性、伤害公式、关卡推荐等级与资源产出设计，用于展示基础数值策划能力。

### 04. 《无畏契约》竞技体验与技能平衡观察

基于 600+ 小时游玩经验，从地图攻防、经济局节奏、英雄技能定位、团队配合与平衡性体验等角度，分析竞技游戏中玩家决策与系统设计之间的关系。
