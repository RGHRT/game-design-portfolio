# 《明日方舟》资源规划与长期养成系统分析

## 项目背景

我是《明日方舟》开服玩家，长期体验游戏的干员养成、材料获取、活动商店、抽卡资源与版本节奏。

这篇分析希望从长期玩家视角出发，观察《明日方舟》如何通过有限资源、阶段性目标和版本运营，让玩家持续形成养成动力。

## 分析重点

- 干员成长系统：等级、精英化、技能专精、模组
- 资源消耗系统：理智、龙门币、经验书、材料、芯片
- 活动商店：短期目标与资源投放
- 抽卡资源：长期预期与玩家规划
- 玩家体验：资源压力、目标感与留存
