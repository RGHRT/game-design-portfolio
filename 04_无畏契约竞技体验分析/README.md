# 《无畏契约》竞技体验与技能平衡观察

## 项目简介

本项目基于本人《无畏契约》600+ 小时游玩体验，从地图攻防、英雄技能、经济系统、团队协作和技能平衡等角度，分析竞技游戏如何通过规则和系统设计影响玩家决策。

相比单纯记录游戏经历，本项目更关注竞技游戏背后的策划逻辑：玩家为什么要控图，为什么经济局会影响连续回合，为什么技能平衡不能只看伤害和胜率。
