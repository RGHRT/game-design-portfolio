# 基于《明日方舟一图流》的玩家资源决策分析

## 项目简介

本项目基于《明日方舟一图流》工具站，结合本人作为《明日方舟》开服玩家的长期体验，分析玩家为什么需要关卡推荐、商店性价比、材料价值表、攒抽计算器等外部工具。

这篇分析希望从玩家实际需求出发，观察《明日方舟》的资源系统、养成系统、活动商店和抽卡预期如何共同影响玩家的长期体验。

- [基于《明日方舟一图流》的玩家资源决策分析](./基于明日方舟一图流的玩家资源决策分析.md)

## 分析重点

- 关卡推荐背后的理智收益与材料效率
- 活动商店性价比与版本资源投放
- 材料价值表与复杂资源系统的统一尺度
- 攒抽计算器与玩家抽卡预期管理
- 外部工具流行对游戏内信息表达的启发
