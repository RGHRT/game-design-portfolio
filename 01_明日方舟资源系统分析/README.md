# 《明日方舟》资源规划与长期养成系统分析

## 项目简介

本项目基于本人作为《明日方舟》开服玩家的长期体验，从干员养成、资源消耗、理智系统、活动商店和抽卡资源规划等角度，分析《明日方舟》如何通过有限资源和阶段性目标维持玩家长期体验。

相比单纯介绍游戏玩法，本项目更关注资源系统背后的策划逻辑：玩家为什么会持续规划养成，为什么会在资源有限的情况下形成目标感，以及这些设计如何影响长期留存。

## 分析重点

- 干员成长链条：等级、精英化、技能专精、模组
- 资源限制：理智、龙门币、经验书、材料、芯片
- 活动商店：短期目标与版本资源投放
- 抽卡资源：未来期待与玩家预期管理
- 玩家体验：资源压力、目标感、满足感与长期留存
